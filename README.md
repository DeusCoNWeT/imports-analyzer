# Imports analyzer
```pre
usage: countImports.py [-h] [-u URL] [-f FILE] [-o OUTPUT] [-s] [-l] [-m]

optional arguments:
  -h, --help            show this help message and exit
  -u URL, --url URL     Introduce the path you want to start parsing
  -f FILE, --file FILE  Introduce the file with all paths you want to start
                        parsing
  -o OUTPUT, --output OUTPUT
                        Introduce the desired output: txt, json.....
  -s, --sorted          Sorts the output
  -l, --listFiles       Lists all files inside the selected directory and sub-
                        directories
  -m, --nomixpanel      Avoid send data to mixpanel
```
